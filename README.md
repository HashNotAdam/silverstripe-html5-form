# SilverStripe HTML5 forms

## Requirements

* SilverStripe 2 or newer
* PHP 5.4+ (using traits)

## Installation

Copy `html5forms` module folder to your SilverStripe installation, run `?flush=all` and done.

## Status

Experimental - not for production use right now ;)
